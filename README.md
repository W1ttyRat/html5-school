# HTML5 School Project

A collection of interactive web pages built with HTML5, CSS3, and JavaScript as part of a web and media course at Tallinn University (Tallinna Ülikool).

🌐 **Live site:** [https://w1ttyrat.github.io/html5-school/](https://w1ttyrat.github.io/html5-school/)

---

## Pages

| Page | Description |
|------|-------------|
| [Home](index.html) | Main landing page with photos, audio/video media, the Gaudeamus hymn, CSS animations, and Bootstrap cards |
| [Hobi](hobi.html) | Hobby page showcasing motocross and shooting videos |
| [Clock](clock.html) | Animated analogue clock built with CSS and JavaScript |
| [Car World](carWorld.html) | Interactive animation of a car, Earth, and Moon with speed controls |
| [Calculator](calculator.html) | A functional calculator supporting basic arithmetic operations |
| [Sudoku](sudoku.html) | Playable Sudoku game with Easy, Medium, Hard, and Expert difficulty levels |
| [Login](login.html) | Login form with email and password fields |
| [Sign Up](signup.html) | User registration form |

---

## Technologies

- **HTML5** – semantic markup and media elements
- **CSS3** – custom styles and animations
- **JavaScript** – interactive features and game logic
- **[Bootstrap 5.3](https://getbootstrap.com/)** – responsive layout and UI components
- **[Font Awesome 6](https://fontawesome.com/)** – icons

---

## Running Locally

No build step is needed. Simply clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/W1ttyRat/html5-school.git
cd html5-school
# Open index.html in your browser
```

Or use a local development server (e.g. VS Code Live Server extension) for the best experience.
